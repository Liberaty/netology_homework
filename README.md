# Netology Homework Collection

Все домашние задания по курсам Netology (DevOps/SRE).

## Структура
- `1. virtualization/` — Виртуализация, Docker, Yandex Cloud
- `2. terraform/`      — Terraform: от основ до Instance Groups
- `3. ansible/`        — Ansible: плейбуки, роли, molecule
- `4. conf-CI-CD/`     — CI/CD: GitLab CI, TeamCity, Nexus
- `5. monitoring/`     — Prometheus, Grafana, ELK
- `6. microservices/`  — Основы микросервисов
- `7. k8s-hw/`         — Kubernetes: от Pod до Helm и Ingress
- `8. org-project/`    — Облачная инфраструктура в Yandex Cloud

## Важно
- Персональные настройки (ключи, пароли) вынесены в `personal.auto.tfvars` — коммитятся только примеры (`*_example`).
- `.tfstate` не хранится в репозитории.
